# Перечень товаров и услуг по классам МКТУ

*Текст для копирования в заявку на регистрацию товарного знака «Вокзал.ТЕХ» (перечень товаров и услуг).*

---

## Класс 9 — Программное обеспечение и носители

Программное обеспечение для автоматизации работы автовокзалов; программное обеспечение для продажи билетов; программное обеспечение для управления расписанием автобусных перевозок; мобильные приложения для покупки билетов на транспорт; программное обеспечение для обработки платежей; электронные билеты; программное обеспечение для сканирования штриховых и QR-кодов; программное обеспечение для диспетчеризации транспорта; компьютерное программное обеспечение для управления базами данных пассажиров; носители с записанным программным обеспечением.

---

## Класс 35 — Услуги в области управления бизнесом

Автоматизация бизнес-процессов автовокзалов; консультационные услуги в сфере организации пассажирских перевозок; услуги по продаже билетов через глобальную компьютерную сеть Интернет; услуги онлайн-бронирования билетов на транспорт; административная обработка заказов на покупку билетов; управление данными о клиентах; составление отчётов в сфере пассажирских перевозок.

---

## Класс 38 — Услуги электросвязи

Предоставление доступа к онлайн-платформе для покупки билетов на транспорт; предоставление доступа к базам данных; услуги электронной почты; передача цифровых файлов; потоковая передача данных; услуги обмена сообщениями; услуги уведомлений через SMS; телекоммуникационные услуги.

---

## Класс 39 — Транспортные услуги

Организация пассажирских перевозок; информация о транспортных услугах; бронирование билетов на транспорт; резервирование мест на транспортные средства; информация о расписании движения транспортных средств; услуги автовокзалов; организация междугородных и пригородных автобусных перевозок.

---

## Класс 42 — Научные и технологические услуги

Разработка программного обеспечения; обновление программного обеспечения; техническая поддержка программного обеспечения; программирование для ЭВМ; консультационные услуги в области компьютерного программного обеспечения; услуги по созданию и поддержке веб-сайтов; платформа как услуга (PaaS); программное обеспечение как услуга (SaaS); облачные вычисления; услуги по проектированию компьютерных систем; услуги по разработке и внедрению информационных систем для автовокзалов.
